GoFo is a start up idea for an application for football playgrounds reservations. In this project we go through
all the software development steps to design a prototype for this application.
Project Includes:
Java Source Code + JavaDoc (GoFo-master)
Software design specifications document(SDS.pdf)
	-Class Diagrams
	-Class Description
	-Sequence Diagram
	-User Interface Design
Mockups for the application UI (GoFo-Design)
